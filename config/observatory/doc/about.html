What are MalariaGEN Analytics? MalariaGEN Analytics is a platform for visualising genomics-based analytics for malaria control. It is based on three core pillars: Open access population genomic dataset on malaria parasites and vectors A set of metrics and analyses derived from these data that are informative about biology and epidemiology Web applications that translates these metrics into actionable information for malaria control and elimination How it works We generate a number of different genetics-based metrics for each individual parasite, from presence/absence of particular alleles to, in the case of drug resistance muations, our assignment of a specific lineage, or independent origin, to each of these alleles. MalariaGEN Analytics presents these metrics in an intuitive and informative way by aggregating information across sites, countries, and regions. Who contributes samples and data to MalariaGEN Analytics? The current data has been contributed by the MalariaGEN network and forms the basis of the latest MalariaGEN community project data release. Information on sites, studies, and researchers can be found within the application. Where can I find out more?
