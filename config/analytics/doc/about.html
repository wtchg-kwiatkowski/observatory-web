About MalariaGEN Analytics MalariaGEN Analytics is a platform for visualising genomics-based analytics for malaria control. It is based on three core pillars: Open access population genomic dataset on malaria parasites and vectors A set of metrics and analyses derived from these data that are informative about biology and epidemiology Web applications that translates these metrics into actionable information for malaria control and elimination How does MalariaGEN Analytics work? We generate a number of different genetics-based metrics for each individual parasite, from presence/absence of particular alleles to, in the case of drug resistance muations, our assignment of a specific lineage, or independent origin, to each of these alleles. MalariaGEN Analytics presents these metrics in an intuitive and informative way by aggregating information across sites, countries, and regions. What data do we use? MalariaGEN Analytics currently includes only those from the full MalariaGEN Pf6.0 dataset that passed quality control. How do we get samples? The MalariaGEN Pf6.0 dataset contains samples collected by collaborators from the MalariaGEN network across the world. Information on sites, studies, and researchers is included within this website. The majority of samples will have been taken as blood draws from people with malaria, from which parasite DNA will have been extracted. However, it is now also possible to extract enough DNA from dried blood spots collected on filter paper to sequence a parasite genome. In the near future, we'll be able to incorporate any available DNA sequence data into the datasets underlying MalariaGEN Analytics. Our vision is that researchers should be able to compare brand new sequences to our database to understand where new infections have come from and what drugs they're resistant to. How do we sequence DNA? We use two technologies to generate genomic data. Wherever possible (depending on having enough DNA) we use whole-genome sequencing technology to read every base in the parasites genome. For samples sequenced at the Wellcome Sanger Institute we use Illumina HiSeq sequencing machines to do this. For samples with low quantity of DNA we concentrate on genotyping roughly 100 informative genetic markers using Illumina MiSeq amplicon sequencing. We then combine all available sequence data into a large united dataset on which we then run our analytics.
